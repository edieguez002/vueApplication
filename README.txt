//---------- FRONTEND ----------//
1.- Install vue -->
npm i vue

2.- Put the folder "/agilProjects" wherever the directory you want and using cmd enter in the folder and execute -->
	# install dependencies
	npm install

	# serve with hot reload at localhost:8080
	npm run dev

	# build for production with minification
	npm run build
	

//---------- BACKEND ----------//
1.- Put "/APICRUD" folder inside xampp "/xampp/htdocs" folder.
2.- Import in phpMyAdmin the file .sql and name the database "agilProjects" with username "agileProjects" and password "agileProjectsTFG".
3.- Run XAMPPS's Apache and SQL.



Finally, run on your browser http://localhost:8080
